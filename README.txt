MOTOMANAGER — PROTOTIPO PREMIUM
================================

Este paquete contiene un prototipo comercial navegable, sin base de datos y sin backend.

CÓMO ABRIRLO
1. Descomprima MotoManager_Prototipo_Premium.zip.
2. Abra la carpeta MotoManager_Prototipo_Premium.
3. Haga doble clic en index.html.
4. Se recomienda Google Chrome o Microsoft Edge.

PANTALLAS INCLUIDAS
- Dashboard ejecutivo
- Agenda visual del taller
- Recepción con responsable y checklist
- Órdenes de trabajo tipo Kanban
- Cotizaciones
- Clientes
- Motocicletas e historial por placa
- Técnicos y recepcionistas
- Inventario con códigos automáticos
- Compras
- Proveedores
- Facturas PDF y pagos
- Reportes con filtros y gráficas
- Exportación Contai en Excel/CSV
- Portal del cliente
- Agendamiento mostrando solo horarios disponibles
- Consulta de estado con técnico y persona que recibió

IMPORTANTE
- Todos los datos son ficticios.
- Las acciones son simuladas.
- No hay persistencia ni base de datos.
- El prototipo funciona sin conexión a Internet.
